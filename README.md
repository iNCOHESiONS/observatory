# Observatory

A small and modular logging library with dynamic formatting capabilities.

![Example](https://i.imgur.com/OTKlH3l.png)

# Quick Start

```python
from observatory import get_logger

logger = get_logger("Example")

logger.info("Hello World!")
```

For more examples, see the [examples folder](https://github.com/iNCOHESiONS/observatory/tree/main/examples).
